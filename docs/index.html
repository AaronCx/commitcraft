CommitCraft

AI-powered CLI that generates conventional commit messages from your git diffs. Stage, run, commit.

TypeScript Node.js Anthropic Claude OpenAI CLI
npm install -g commitcraft

Features

Multi-Provider AI

Switch between Anthropic Claude and OpenAI with a single flag. Use the model that fits your workflow.

Conventional Commits

Every message follows the Conventional Commits spec with proper types, scopes, and imperative mood.

Clipboard & Auto-Commit

Copy to clipboard with --copy or commit directly with --commit. Confirm before any destructive action.

Smart Diff Handling

Automatically captures staged changes, falls back to unstaged, and truncates large diffs to stay within token limits.

Persistent Config

XDG-compliant config storage for API keys and preferences. Set once, use everywhere.

Developer Experience

Colored output, spinners, friendly error messages, and --dry-run for previewing diffs before sending.
